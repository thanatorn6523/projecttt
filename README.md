-----การใช้งานเว็บ Shop shop----- 
1.downloadโค้ดจากในgithub 
2.เปิดterminal ในvscode 
3.พิมcd myproject ในterminal
4.พิมpython manage.py runsserver
5.กดที่http://127.0.0.1:8000/
จะเข้าสู่หน้าเว็บShop Shop
--------------------------------------
วิธีการเข้าไปเพื่มลบสินค้า
เข้าไปในhttp://127.0.0.1:8000/admin/
สามารถกดเข้าไปเพื่มลบตั้งราคาสินค้าได้ มีมหวดหมู๋สินค้า
